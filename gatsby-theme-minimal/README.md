## What's in the box?

- [x] TypeScript
- [x] Yarn workspaces
- [x] Minimal Gatsby Theme
- [x] Demo site
- [x] ESLint + Prettier + Husky

## Quick Start

```sh
$ npx gatsby new my-theme https://github.com/epilande/gatsby-starter-theme-ts
$ cd my-theme
```

### Start demo dev environment

```sh
$ yarn dev
```
